### Getting Started

#### Prerequisites
- Python 3.x
- Selenium WebDriver
- Required Python packages (see requirements or imports in the test script)

#### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Imesh-Bandara/ITPM-Assignment-1.git
   cd ITPM-Assignment-1
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

#### Running Tests
Execute the test automation script:
```bash
python IT_23313316test_automation.py
```
